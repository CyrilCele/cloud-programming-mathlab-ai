# Infrastructure Tests

This directory contains documentation and supporting materials for infrastructure verification.

## Purpose

Infrastructure testing verifies that deployed AWS resources function correctly after Terraform provisions them.

Future milestones will include verification procedures for:

- VPC
- Subnets
- Security Groups
- IAM Roles
- EC2 Instances
- Launch Template
- Application Load Balancer
- Auto Scaling Group
- Amazon S3
- Amazon CloudFront
- Amazon Route 53
- Amazon CloudWatch

## Current Status

Infrastructure testing will begin once AWS resources are provisioned in later milestones.

At Milestone 1, no AWS infrastructure has been deployed.
