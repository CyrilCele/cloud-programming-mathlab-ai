# Support

## Overview

This project was developed as part of a university Infrastructure-as-Code assignment.

Support is provided on a best-effort basis.

## Before Requesting Support

Please ensure that you have:

- Read the documentation.
- Run the validation scripts.
- Verified your AWS credentials.
- Confirmed your Terraform version.

## Troubleshooting Resources

- docs/TROUBLESHOOTING.md
- docs/DEPLOYMENT.md
- docs/TESTING.md

## Reporting Issues

Include:

- Operating system
- Terraform version
- AWS CLI version
- Error output
- Relevant logs
